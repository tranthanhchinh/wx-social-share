== Description ==

Wx Social Share help you share post to social.Better seo support.

== Installation ==

1. Upload `wx-social-share` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place settings option Settings->WX Social Share display in your templates
